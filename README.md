# ⚖️ Lawyer Website (Laravel)

## 📌 Description
A professional lawyer website developed using Laravel and PHP. The platform provides information about legal services, allows users to contact lawyers, and supports online consultation requests.

## 🚀 Features
- Home Page with Legal Services
- About Lawyer / Law Firm
- Contact & Consultation Form
- Client Inquiry System
- Responsive UI Design
- Admin Panel (optional)

## 🛠️ Technologies Used
- Laravel (PHP Framework)
- MySQL
- HTML, CSS, JavaScript
- Bootstrap
